# 内存操作优化
## 建议
1. 展开读取内存的循环
2. 消除数据相关性
3. 同时向存储控制器发送多个查询指令
4. 请求按不少于32个字节的增量方式读取数据
5. 使用所有经过请求的页面
6. 以一种排除了对相同 DRAM 页面进行选取的增量方式来处理数据
7. 对齐数据源地址 
8. 组合执行存取内存的代码 
9. 成组进行读写操作 
10. 仅仅在必要时才访问内存
11. 从不针对特定平台优化代码
